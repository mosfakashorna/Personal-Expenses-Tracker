# Personal Expense Tracker

This is a simple command-line expense tracker built with Python.
This program allows users to record expenses by category and view a summary of their spending.

## Features
- Add expenses by category
- Automatically adds expenses of the same category
- View total spending by category.
- Display the total amount spent.
- Clear formatted output for readability

# How It Works

This program asks the user to record expenses by category and store the data inside a Python dictionary.

If user records same category multiple times, the program adds automatically.
when users want to see their spending summary, the program displays a clear output for readability.

# How to run 
- Make sure Python is installed on your computer.
- Download or clone this repository.
- Open the terminal in the project folder.
- Run the program: Python Personal-Expense-Tracker.py
- Follow the manu options shown in the terminal
